
## Install Cygwin without Internet connectivity.

`install.cmd`

This command will use pre-downloaded Cygwin content from `distrib`
directory prepared by `update.cmd` for offline installation.

The command is starts Cygwin setup in unattendant mode.

## Update Cygwin from Internet.

`update.cmd`

This command will pre-download Cygwin content in `distrib` directory
and create a zip archive for offline installation by `install.cmd`.

The command is starts Cygwin setup in interactive mode.

## Configure list of Cygwin packages.

See `package.conf`.

## Features.

* No dependencies on software external to Windows.
* Automatic detection of 32/64 bit architectures.
* Simple straightforward workflow.
